# Assignment 4: Reinforcent Learning
This README can be used to understand how to run analyses for assignment 4 in CS 7641. To be able to properly run these files, you must have Python hiive-mdptoolbox, and gym installed.
## Directory Overview
- **plots**
    - The plots directory includes png images of every figure generated for the report.
## Entrypoints
- forest.py
    - This file is the main entrypoint for running the the RL analysis on the forest MDP from the MDPToolbox kit.
    - To run this file, simply run
    ```python
    python forest.py
    ```
- frozen_lake.py
    - This file is the main entrypoint for running the RL analysis on the frozen lake MDP from the Open-AI gym module.
    - To run this file, simply run
    ```python
    python frozen_lake.py
    ```
## Helper Files
- utils.py contains code for performing certain functions shared between problems or algorithms.
